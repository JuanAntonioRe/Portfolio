About me

Hi, I'm Juan Antonio. I am a mechanical engineer specialized in Data Science with experience in data analysis in manufacturing areas. My professional career is focused on production areas and laboratory testing. I graduated from TripleTen data science bootcamp and I'm eager to apply my Python skills to improve and optimize analytical processes by implementing Artificial Intelligence projects.

I found my passion for data when I was working at Manufacturas Kaltex, I had to analyze the efficiency of the machines using Excel and the feeling that I felt when I discover the problem with my analisys, was so amazing. I realized of the power that data have to transform and solve problems. Since then, I've being learning avanced techniques of data analysis.

Now, I'm able to use machine learning techniques in order to make predictions, some of those techniques are Python, pandas, scikit-learn and SQL. I have differents projects of data science and my goal is to keep growing in this passionate and exciting field

Projects

Bootcamp final project

The objective is to forecast the customer cancellation rate of the Interconnect company. This way, if a use plans to leave, they will be offered special plan options. In order to accomplish the task, Interconnect's marketing team has collected some of its customers' personal data, including information about their plans and contracts. For this project it has been established that the key metric is AUC-ROC.

Artificial Vision

Supermarket chain Good Seed would like to explore whether data science can help it comply with alcohol laws by ensuring it does not sell alcohol to underage people.

  • Stores are equipped with cameras in the checkout area, which are activated when a person is purchasing alcohol.
  • Computer vision methods can be used to determine a person's age from a photo.

The task, then, is to build and evaluate a model to verify the age of people. The project trains a model that is capable of predicting the age of a person through an image. It has been requested that the MAE have a value less than 8.

Visit the repository at GitHub

Time series

A Taxi company called Sweet Lift has collected historical data on taxi orders at airports. To attract more drivers during peak hours, you need to predict the number of taxi orders for the next hour. The RECM metric on the test set should not be greater than 48. The data must be resampled so that each data point of the original data falls within one-hour intervals.

Visit the repository at GitHub

Car price

Rusty Bargain is a second-hand car sales service that is developing an app to attract new customers. Thanks to this app, you can quickly find out the market value of your car. The dataset has information on technical specifications, equipment versions, prices, among other data. For this project, gradient descent and gradient boosting algorithms are used.

Visit the repository at GitHub
